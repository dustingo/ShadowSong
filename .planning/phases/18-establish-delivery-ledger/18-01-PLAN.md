## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| webhook send path → delivery ledger | 发送链路把运行时上下文和失败结果持久化到 PostgreSQL，任何字段污染都会进入后续审计真源 |
| delivery ledger → operator read surface | 持久化快照会被维护者查询，若快照范围失控会泄露敏感配置 |
